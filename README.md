# kingreading 国王阅读
### 简介
* 像国王一样享受私人内容整理助理的服务。
* 智能整理阅读内容,定时终端内容推送。
* 支持多种阅读内容来源，小说、博客、实时资讯
* 该系统将需要阅读的内容，自动根据设定的时间和方式，根据阅读终端进行推送。
